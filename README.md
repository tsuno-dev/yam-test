
# TSV First Normal Form Converter

TSV（タブ区切り）形式のデータを読み込み、「第1正規化」およびその「逆変換（集約）」を行うツールです。

## 1. 概要

本プログラムは、以下の2つの機能を提供します。

* 
**第1正規化**: 1つのセルにコロン（`:`）で区切られた複数の値が含まれるTSVを読み込み、それぞれの値を個別の行に展開して出力します。


* 
**第1正規化の逆変換**: 同じキーを持つ複数の行を読み込み、値をコロン（`:`）で連結して1行に集約します。



## 2. 動作環境

**Java**: Java 17 (OpenJDK 11以上の環境で動作可能) 

* **ビルドツール**: Maven

## 3. プロジェクト構造

```text
.
├── pom.xml
├── .gitignore
├── input1.tsv（第1正規化テスト用TSVファイル）
├── input2.tsv（逆変換テスト用TSVファイル）
└── src
    └── main
        └── java
            └── jp
                └── assignment
                    ├── TsvFirstNormalFormConverter.java  (メイン・入出力制御)
                    └── FirstNormalizationService.java     (第1正規化または逆変換の機能を実装)

```

## 4. ビルド方法

プロジェクトのルートディレクトリ（`pom.xml` がある場所）で以下のコマンドを実行してください。

```bash
mvn clean package

```

## 5. 実行方法

本プログラムは、実行後に標準入力から「実行モード」と「処理対象のデータ」を入力します。
動作確認用のサンプルとして、プロジェクトのルートディレクトリに input.tsv を配置しています。

### 起動コマンド

```bash
java -cp target/classes jp.assignment.main.TsvFirstNormalFormConverter

```

### 実行手順

1. プログラムを起動するとモード選択の入力待ちになります 。

2. 
**第1正規化**を行う場合は `1`、**逆変換**を行う場合は `2` と入力し、Enterを押してください。

3. 続けて、処理したいTSVデータを貼り付けてください。

4. 続けて、出力したいTSVデータのファイル名を指定してください。

5. 「第1正規化」または「逆変換」されたTSVデータが出力されます。
